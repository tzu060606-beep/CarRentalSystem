## 詳細使用說明

### 1. BaseModal.vue — 通用 Modal 外框

```vue
<script setup>
import BaseModal from '@/components/common/BaseModal.vue'
const showModal = ref(false)
</script>

<template>
  <button @click="showModal = true">開啟</button>

  <BaseModal :isVisible="showModal" title="新增資料" size="lg" @close="showModal = false">
    <template #body>
      <!-- 放表單內容 -->
    </template>
    <template #footer>
      <button class="btn btn-outline-secondary" @click="showModal = false">取消</button>
      <button class="btn btn-primary" @click="handleSubmit">確認</button>
    </template>
  </BaseModal>
</template>
```

**Props：**
| 名稱 | 型別 | 預設 | 說明 |
|---|---|---|---|
| isVisible | Boolean | 必填 | 控制顯示/隱藏 |
| title | String | '' | 標題文字 |
| size | String | 'md' | sm / md / lg / xl |

**Events：** `@close`

---

### 2. ConfirmDialog.vue — 確認對話框

```vue
<ConfirmDialog
  :isVisible="showConfirm"
  title="確認刪除"
  message="確定要刪除這筆資料嗎？此操作無法還原。"
  confirmText="刪除"
  confirmVariant="danger"
  @confirm="handleConfirm"
  @cancel="showConfirm = false"
/>
```

**Props：**
| 名稱 | 型別 | 預設 | 說明 |
|---|---|---|---|
| isVisible | Boolean | 必填 | 控制顯示/隱藏 |
| title | String | '確認操作' | 標題 |
| message | String | '確定要執行此操作嗎？' | 說明文字 |
| confirmText | String | '確認' | 確認按鈕文字 |
| cancelText | String | '取消' | 取消按鈕文字 |
| confirmVariant | String | 'primary' | primary / danger / warning |

**Events：** `@confirm` / `@cancel`

---

### 3. AlertToast.vue — 成功/失敗提示

```vue
<script setup>
import AlertToast from '@/components/common/AlertToast.vue'
const toast = ref(null)
</script>

<template>
  <AlertToast ref="toast" />
  <button @click="toast.show('儲存成功！', 'success')">儲存</button>
</template>
```

**show(message, type, duration) 參數：**
| 參數 | 說明 | 預設 |
|---|---|---|
| message | 顯示文字 | 必填 |
| type | success / danger / warning / info | 'success' |
| duration | 顯示毫秒數 | 3000 |

**注意：** template 裡不需要 `.value`，直接用 `toast.show()`

### 4. LoadingSpinner.vue — 載入中


**Props：**
| 名稱 | 型別 | 預設 | 說明 |
|---|---|---|---|
| isLoading | Boolean | false | 控制顯示 |
| text | String | '載入中...' | 文字 |
| overlay | Boolean | false | 全頁遮罩模式 |

### 6. ActionButtons.vue — 操作按鈕組

```vue
<!-- 完整版（編輯+刪除） -->
<ActionButtons @edit="handleEdit(item.id)" @delete="handleDelete(item.id)" />

<!-- 只顯示編輯 -->
<ActionButtons :showDelete="false" @edit="handleEdit(item.id)" />
```

**Props：** `showEdit`（預設 true）/ `showDelete`（預設 true）/ `editText` / `deleteText`
**Events：** `@edit` / `@delete`

---

### 7. SearchBar.vue — 搜尋列

```vue
<!-- 有搜尋按鈕 -->
<SearchBar v-model="keyword" @search="handleSearch" placeholder="請輸入關鍵字..." />

<!-- 即時搜尋（不需要按鈕，配合 computed 使用） -->
<SearchBar v-model="keyword" :showButton="false" />
```

**Props：** `modelValue` / `placeholder` / `showButton`（預設 true）
**Events：** `@search`

### 8. Pagination.vue — 分頁

**Props：** `currentPage`（必填）/ `totalPages`（必填）/ `maxVisible`（預設 5）
**Events：** `@change`（回傳頁碼）

### 9. StatusBadge.vue — 狀態標籤

```vue
<!-- 使用預設 map -->
<StatusBadge status="ACTIVE" />
<StatusBadge status="CANCELLED" />

<!-- 自訂 map（點數模組專用） -->
<StatusBadge :status="order.orderStatus" :map="{
  ACTIVE:    { label: '待使用', variant: 'primary' },
  USED:      { label: '已使用', variant: 'success' },
  CANCELLED: { label: '已取消', variant: 'secondary' },
  EXPIRED:   { label: '已過期', variant: 'warning' }
}" />
```

**預設支援：** ACTIVE / INACTIVE / PENDING / CANCELLED / COMPLETED / CLOSED

```vue
<Breadcrumb :items="[
```

### 14. ImageUploader.vue — 圖片上傳

```vue
<script setup>
const imageBase64 = ref('')
</script>

<template>
  <!-- 新增時 -->
  <ImageUploader v-model="imageBase64" />

  <!-- 修改時（帶入現有圖片） -->
  <ImageUploader v-model="imageBase64" :previewUrl="product.imageUrl" />
</template>
```

**Props：**
| 名稱 | 型別 | 預設 | 說明 |
|---|---|---|---|
| modelValue | String | '' | base64 字串（v-model 綁定） |
| previewUrl | String | '' | 現有圖片 URL（修改時使用） |
| accept | String | 'image/*' | 接受的檔案類型 |
| maxSizeMB | Number | 5 | 最大檔案大小（MB） |

**說明：** v-model 回傳 base64 字串，後端需要自行處理儲存。

## ⚠️ 注意事項

1. **Font Awesome 已在 main.js 全域註冊**，直接使用 `<font-awesome-icon icon="圖示名稱" />` 即可，不需要在每個元件 import。

2. **使用 Solid 系列圖示**（`fas`），例如 `star`、`inbox`、`arrow-left`。
   查詢圖示名稱：https://fontawesome.com/icons?s=solid

3. **BaseModal 和 ConfirmDialog 使用 Teleport**，會渲染到 `<body>` 最底部，不受父元素 z-index 影響。

4. **AlertToast 需要用 ref 取得實例**，在 template 裡呼叫 `toast.show()`（不需要 `.value`）。

5. **ImageUploader 回傳 base64**，後端需要自行處理轉換和儲存。
